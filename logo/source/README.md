## Farbvarianten

### 1. Primary (Orange auf Schwarz)
- **Flamme & Bogen:** `#F47A20` (Sauna-Orange)
- **Text:** `#FFFFFF` (Weiss)
- **Hintergrund:** Transparent/Schwarz
- **Anwendung:** Fasssaunen, Fahrzeugbeschriftung, Merchandise auf dunklem Grund

### 2. Light (Orange/Anthrazit auf Weiss)
- **Flamme:** `#F47A20` (Sauna-Orange)
- **Bogen & Text:** `#3D3D3D` (Anthrazit)
- **Hintergrund:** Transparent/Weiss
- **Anwendung:** Website, Briefpapier, helle Drucksachen

### 3. Monochrome (Schwarz auf Weiss)
- **Alle Elemente:** `#000000` (Schwarz)
- **Hintergrund:** Transparent/Weiss
- **Anwendung:** Stickerei, Gravur, Stempel, 1-Farb-Druck

### 4. Negative (Weiss auf Schwarz)
- **Alle Elemente:** `#FFFFFF` (Weiss)
- **Hintergrund:** Transparent/Schwarz
- **Anwendung:** Dunkle Textilien, Negativ-Praegung, dunkle Holzflaechen

## Logo-Elemente

Das Logo besteht aus:
1. **Halbkreis/Bogen** - Symbolisiert die Fasssauna-Form
2. **Flamme mit negativem S** - Steht fuer Waerme und den Anfangsbuchstaben
3. **Horizontale Linie** - Verbindet Bogen und Text
4. **Wortmarke SAUNAZEIT** - In Montserrat SemiBold, Versalien

## Technische Spezifikationen

- **Format:** SVG (Vektor, skalierbar)
- **Viewbox:** 400 x 300 px
- **Schrift:** Montserrat SemiBold (Google Fonts)
- **Zeichensatz:** UTF-8

## Hinweise

- SVG-Dateien koennen in Adobe Illustrator geoeffnet und bearbeitet werden
- Fuer Web-Export: SVG direkt verwenden oder nach PNG/WebP konvertieren
- Fuer Print: Nach PDF/EPS exportieren, Schriften in Pfade umwandeln
- Mindestgroesse fuer Lesbarkeit: 80px Breite (digital), 25mm (Print)
